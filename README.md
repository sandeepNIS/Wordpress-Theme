<img width="1351" height="625" alt="image" src="https://github.com/user-attachments/assets/6e7c303c-ddd0-4819-88ce-7c3740391b43" /># Nichi Modern

A lightweight, modern WordPress theme built with clean PHP and responsive CSS.

## Project Structure

```
nichi-modern/
├── index.php           # Main template file
├── header.php          # Header template
├── footer.php          # Footer template
├── functions.php       # Theme functions and hooks
├── style.css           # Main stylesheet (theme header)
├── assets/
│   ├── css/
│   │   └── main.css    # Primary stylesheet
│   └── js/
│       └── main.js     # JavaScript functionality
└── README.md          # This file
```

## Features

- Clean, semantic PHP code
- Responsive design
- Modular template structure
- Asset management (CSS/JS)

## Installation

1. Clone or download the theme to your WordPress themes directory:

   ```
   wp-content/themes/nichi-modern/
   ```

2. Activate the theme in WordPress Admin > Appearance > Themes

## File Descriptions

- **index.php** - The main template file that handles all page rendering
- **header.php** - Contains the HTML `<head>` section and opening page markup
- **footer.php** - Contains closing markup and footer content
- **functions.php** - Registers theme features, hooks, and enqueues assets
- **style.css** - Theme header file (required for WordPress theme identification)
- **assets/css/main.css** - Primary stylesheet with all theme styling
- **assets/js/main.js** - Custom JavaScript for theme functionality

## Usage

Edit the template files to customize:

- Site structure and layout
- Header content
- Footer content
- Add hooks in `functions.php` for additional functionality

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher

## Development

To modify the theme:

1. Edit template files directly
2. Update styles in `assets/css/main.css`
3. Add JavaScript functionality in `assets/js/main.js`
4. Register custom functions in `functions.php`
